LeetCode每日一题 2025-02-07

February 6, 2025 · 1 min

Today’s Question#

https://leetcode.com/problems/find-the-number-of-distinct-colors-among-the-balls/description

Solution Approach#

We only need to know the number of available colors in the current list. To do this, we need to keep track of the number of balls of each color. If the number of balls of a certain color becomes zero, then col_cnt -= 1. If a new color appears, then col_cnt += 1.

Methods#

  • Maintain a hashtable for each ball and each color.
  • Modify the color of a certain ball.
  • If the number of balls of that color becomes zero, then col_cnt -= 1.
  • If a new color appears, then col_cnt += 1.

Complexity#

  • Time complexity: $O(N)$, where N is the length of the query array.
  • Space complexity: $O(N)$, where N is the length of the query array.

Potential Advanced Questions#

What if we now need to change the colors of balls within a specific range? For example, given inputs (x, y, z), change the color of all balls whose IDs are between x and y. How many balls now have different colors?
