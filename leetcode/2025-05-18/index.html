1931. A solution that uses three different colors to color the grid.
May 18, 2025 · 2 min

Today’s Problem#

https://leetcode.com/problems/painting-a-grid-with-three-different-colors/

Idea#

In this problem, we notice that $m$ is much smaller than $n$. Since $m \le 5$ and $n \le 1000$, we can enumerate all valid coloring patterns for each column, and then generalize this to the entire matrix. During this process, we discover a key property: no matter how a column is colored, its influence only affects the adjacent column, not the columns further away. This property makes dynamic programming possible.

Method#

Based on the above analysis, our solution steps are as follows:

  1. First, find all valid coloring patterns for each column. This is done by using depth-first search to enumerate all possible combinations.
  2. Next, determine which two patterns can appear in adjacent columns. We check each pair of patterns to see if they satisfy the constraints for adjacent columns.
  3. Then, use dynamic programming to move from one column to the next. The DP transition equation is: $DP[col][case_x] = \sum DP[col-1][case_y]$, where $\forall \text{case_x}$ and $\text{case_y}$ can appear in adjacent columns.
  4. Finally, sum up the values of all cases in the last column of DP to get the answer.

Complexity#

  • Time complexity: $O(3^{2m} \times n)$
  • Space complexity: $O(3^{2m})$

Code#

class Solution:
    def colorTheGrid(self, m: int, n: int) -> int:
        pat = []
        col = [0, 1, 2]
        def dfs(x, s):
            if x == m:
                pat.append(s)
                return
            for i in col:
                if x == 0 or s[x - 1] != i:
                    dfs(x + 1, s + [i])
        dfs(0, [])

        # till this step, we find all valid patterns for a column and store it in the pattern list.

        l = len(pat)
        valid = [[True for _ in range(l)] for _ in range(l)]

        for i in range(l):
            for j in range(i + 1, l):
                for k in range(m):
                    if pat[i][k] == pat[j][k]:
                        valid[i][j] = False
                        break
        
        # till this step, we find all the pattern pairs that is valid.

        dp = [[0 for _ in range(l)] for _ in range(n)]
        mod = 1_000_000_007
        for i in range(l):
            dp[0][i] = 1

        # for column 0, each pattern is valid.

        for i in range(1, n):
            for x in range(l):
                for y in range(x + 1, l):
                    if valid[x][y]:
                        dp[i][x] = (dp[i][x] + dp[i-1][y]) % mod
                        dp[i][y] = (dp[i][y] + dp[i-1][x]) % mod
        
        # we elaborate to the next column according to the DP formula.

        ans = 0
        for i in range(l):
            ans = (ans + dp[-1][i]) % mod

        # finally, we add up all the answers.

        return ans
