January 6, 2025 · 2 min

Today’s Problem#

https://leetcode.com/problems/minimum-number-of-operations-to-move-all-balls-to-each-box/description

Solution Approach#

For each position, it’s easy to see that the answer at that position is equal to the sum of the distances from i to all 1s. That is: $ans[i]=\sum_{j=0}^{n-1} boxes[j] \times abs(j-i)$. Now, imagine a pointer moving from left to right (from 1 to n), calculating the result step by step. It can be observed that the difference between $ans[i]$ and $ans[i+1]$ is equal to the number of 1s between 1 and i minus the number of 1s between i+1 and n. That is: $ans[i+1] - ans[i] = \sum_{j=0}^{i} boxes[j] - \sum_{j=i+1}^{n-1} boxes[j]$. Therefore, by calculating the number of 1s on the left of i and the total number of 1s in the sequence, we can find all answers in $O(N)$ time.

Algorithm Steps#

First, use the formula $ans[0]=\sum_{j=1}^{n} boxes[j] \times j$ to calculate $ans[0]$ and the total number of 1s in the sequence. This requires iterating through the entire sequence. Next, for each i (from 1 to n-1), use the formula $ans[i+1] - ans[i] = \sum_{j=0}^{i} boxes[j] - \sum_{j=i+1}^{n-1} boxes[j]$ to compute the value recursively. Since $ans[i+1] - ans[i] = \sum_{j=1}^{i} boxes[j] - \sum_{j=i+1}^{n} boxes[j] = 2 \times \sum_{j=0}^{i} boxes[j] - \sum_{j=0}^{n-1} boxes[j]$, we can maintain a count of 1s on the left during the iteration, and then apply the above formula.

Complexity Analysis#

  • Time complexity: $O(N)$, where N is the length of boxes.
  • Space complexity: $O(N)$, used for storing the answer array.

Code#

class Solution:
    def minOperations(self, boxes: str) -> List[int]:
        now = 0
        cnt1 = 0
        for i in range(len(boxes)):
            if boxes[i] == '1':
                now += i
                cnt1 += 1

        ans = [now] * len(boxes)

        now_cnt1 = 0
        for i in range(1, len(boxes)):
            if boxes[i-1] == '1':
                now_cnt1 += 1
            ans[i] = ans[i-1] + 2 * now_cnt1 - cnt1
        
        return ans
