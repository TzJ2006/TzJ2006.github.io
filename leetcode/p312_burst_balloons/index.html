P312 Explode balloons

P.312 Solutions to the problems
September 6, 2025 · 2 min

Problem#

Pop the Balloon

Intuition#

When popping a balloon, it does not affect the scores of other balloons, and the length of the list decreases continuously. This means we can use dynamic programming to solve this problem.

Approach#

Consider the following definition: $dp[i][j]$ represents the maximum score that can be obtained by popping all balloons between i and j. In this case, if n is the length of the list, then the answer is $dp[0][n-1]$.

Now let’s consider how to calculate $dp[i][j]$. As mentioned in Intuition, the number of balloons decreases continuously, so we can first calculate the results for shorter intervals, and then gradually extend the intervals to $0 \to n-1$.

Our solution is as follows:

First, iterate through the interval lengths.

Next, split the interval $i \to j$ into two sub-intervals: $i \to k-1$ and $k+1 \to j$.

Then, add the scores of the sub-intervals $i \to k-1$ and $k+1 \to j$, plus the score of popping the kth balloon, to get the total score of the interval $i \to j$.

Since the balloons in sub-intervals $i \to k-1$ and $k+1 \to j$ have already been popped, the increment of the score for popping the kth balloon is $nums[k] \times nums[i-1] \times nums[j+1]$.

Finally, note that the boundaries of the sub-intervals need to be handled carefully: since the current balloon being popped could be at the boundary of the interval $i \to j$, we need to iterate over k within the closed interval $[i, j]$.

The final answer is thus $dp[0][n-1]$.

Complexity#

  • Time complexity: $O(N^3)$, where N is the length of the list.
  • Space complexity: $O(N^2)$, where N is the length of the list.

Code#

class Solution:
    def maxCoins(self, nums: List[int]) -> int:
        n = len(nums)
        nums = nums + [1]
        dp = [[0 for _ in range(n+1)] for _ in range(n+1)]

        for i in range(n):
            dp[i][i] = nums[i] * nums[i-1] * nums[i+1]

        for i in range(1, n):
            for l in range(n):
                if i + l == n:
                    break
                r = i + l
                for k in range(l, r+1):
                    dp[l][r] = max(dp[l][r], dp[l][k-1] + dp[k+1][r] + nums[k] * nums[l-1] * nums[r + 1])

        # print(dp)

        return dp[0][n-1]
