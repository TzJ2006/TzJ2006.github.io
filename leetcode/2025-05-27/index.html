2894. 如何求解可以被整除的数字与不能被整除的数字之间的差值？
May 27, 2025 · 2 min

Today’s Problem#

2894. Difference between Divisible and Non-divisible Sums

Solution Approach#

Given an integer n and a divisor m, we need to calculate the difference between:

  1. The sum of all numbers from 1 to n that are not divisible by m.
  2. The sum of all numbers from 1 to n that are divisible by m.

We then divide the numbers from 1 to n into two groups based on whether they are divisible by m, and calculate the sum of each group before returning the difference.

Solution Methods#

There are two ways to solve this problem:

Method 1: Formula-based Approach#

  • Use the formula for the sum of the first n natural numbers: n * (n + 1) // 2 to find the total sum.
  • Calculate the number of numbers from 1 to n that are divisible by [[[HYMTPH_19]]: k = n // m.
  • The numbers that are divisible are: m, 2m, ..., km, and their sum is [[[HYMTPH_22]].
  • Subtract the sum of the divisible numbers from the total sum to get the sum of the non-divisible numbers, and then find the difference.

Method 2: Brute-force Enumeration#

  • Iterate through all numbers from 1 to n.
  • If a number is divisible by m, add it to [[[HYMTPH_26]].
  • Otherwise, add it to [[[HYMTPH_27]].
  • Return the difference, which is num1 - num2.

Complexity Analysis#

  • Time complexity:

    • Method 1: $O(1)$ (using formulas, constant time)
    • Method 2: $O(n)$ (linear iteration)

  • Space complexity:

    • Both methods are $O(1)$ (only using a few variables)

Code#

class Solution:
    def differenceOfSums(self, n: int, m: int) -> int:
        # Method 1: Formula-Based
        total_sum = n * (n + 1) // 2
        k = n // m
        divisible_sum = m * (k * (k + 1) // 2)
        return total_sum - divisible_sum

class Solution:
    def differenceOfSums(self, n: int, m: int) -> int:
        # Method 2: Brute-Force Iteration
        num1, num2 = 0, 0
        for i in range(1, n + 1):
            if i % m == 0:
                num2 += i
            else:
                num1 += i
        return num1 - num2
