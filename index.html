Thu, Jan 2, 2025 Today’s problem https://leetcode.com/problems/number-of-ways-to-split-array/description/ Intuition We need to calculate $sum(a[0]~to~a[i])$ and $sum(a[i+1]~to~a[n])$ according to the question. Since $sum(a[0]~to~a[i + 1]) = sum(a[0]~to~a[i]) + a[i+1]$ and $sum(a[i+1]~to~a[n]) = sum(a[0]~to~a[n]) - sum(a[0]~to~a[i])$. Therefore, we can new two variables. One $now$ that stores $sum(a[0]~to~a[i])$ and add $a[i+1]$ to it every iteration, one $summ$ that stores $sum(a[0]~to~a[n])$. Approach In this case, we only need to compare $now$ and $summ - now$.
