2894. 如何找出那些能被某个数整除的数字与那些不能被该数整除的数字之间的区别？
2025年5月27日 · 1 分钟

今日问题#

2894. 可除数与不可除数的差值

解题思路#

给定一个整数[[[HYMTPH_4]]和一个除数m，我们需要计算以下两个数值的差值：

  1. 从1到[[[HYMTPH_7]]的所有数中，不是能被[[[HYMTPH_8]]整除的数的和。
  2. 从1到[[[HYMTPH_10]]中所有能被[[[HYMTPH_11]]整除的数的和。

然后，我们将从1到[[[HYMTPH_13]]的数分为两组，根据它们是否能被[[[HYMTPH_14]]整除，并计算每组的求和，最后得到差值。

解题方法#

解决这个问题有两种方法：

方法1：基于公式的方法#

  • 使用前[[[HYMTPH_15]]个自然数的求和公式：n * (n + 1) // 2来计算总和。
  • 计算从[[[HYMTPH_17]]到[[[HYMTPH_18]]中能被k = n // m整除的数的数量：k = n // m。
  • 能被整除的数为：m, 2m, ..., km，它们的和为[[[HYMTPH_22]]。
  • 从总和中减去能被整除的数的和，得到不可除数的和，然后求出差值。

方法2：暴力枚举法#

  • 遍历从[[[HYMTPH_21]]到[[[HYMTPH_22]]的所有数。
  • 如果一个数能被[[[HYMTPH_23]]整除，则将其加到[[[HYMTPH_26]]中。
  • 否则，将其加到[[[HYMTPH_27]]中。
  • 返回差值，即[[[HYMTPH_24]]。

复杂度分析#

  • 时间复杂度：

    • 方法1：$O(1)$（使用公式，常数时间）
    • 方法2：$O(n)$（线性迭代）

  • 空间复杂度：

    • 两种方法都是$O(1)$（只使用少量变量）

代码#

class Solution:
    def differenceOfSums(self, n: int, m: int) -> int:
        # Method 1: Formula-Based
        total_sum = n * (n + 1) // 2
        k = n // m
        divisible_sum = m * (k * (k + 1) // 2)
        return total_sum - divisible_sum

class Solution:
    def differenceOfSums(self, n: int, m: int) -> int:
        # Method 2: Brute-Force Iteration
        num1, num2 = 0, 0
        for i in range(1, n + 1):
            if i % m == 0:
                num2 += i
            else:
                num1 += i
        return num1 - num2
