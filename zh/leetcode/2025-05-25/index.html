2131. 如何找到一个由单词中的两个字母连接而成的最长回文字符串？解决方法。
2025年5月25日 · 1 分钟

今日题目#

2131. 通过连接两个字母的单词得到最长的回文字符串

思路#

形成回文字符串有两种方式：

  1. 某个字符串在列表中作为其反转字符串存在。
  2. 某个字符串本身就是一个回文串。

其中，本身就是回文的字符串只能放在整个回文串的正中间。

解决方法#

因此，我们可以使用哈希表来解决这个问题。 注意：首先检查第一种情况，然后检查第二种情况。 如果某个字符串的反转字符串在列表中存在，则将这两个字符串都添加到结果中。

然后，检查该字符串本身是否为回文串。

复杂度#

  • 时间复杂度：

$O(N)$，其中N是单词的长度。

  • 空间复杂度：

$O(N)$，其中N是单词的长度。

代码#

class Solution:
    def longestPalindrome(self, words: List[str]) -> int:
        cnt = Counter(words)
        ans = 0
        sp = 0
        for word, t in cnt.items():
            # print(word, t)
            if word[0] == word[1]:
                ans += (t - t % 2)
                sp |= (t % 2)
            else:
                ans += min(t, cnt[word[::-1]])

        return (ans + sp) * 2
