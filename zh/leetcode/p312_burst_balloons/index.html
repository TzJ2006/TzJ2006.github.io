P312 爆破气球

第312页 问题的解决方案
2025年9月6日 · 1 分钟

问题#

弹出气球

直觉#

在弹出气球的过程中，其他气球的得分不会受到影响，而且列表的长度会持续减少。这意味着我们可以使用动态规划来解决这个问题。

方法#

定义如下：$dp[i][j]$表示通过弹出第i个到第j个气球可以获得的最大分数。在这种情况下，如果n是列表的长度，那么答案就是$dp[0][n-1]$。

现在我们来考虑如何计算$dp[i][j]$。如直觉所述，气球的数量会持续减少，因此我们可以先计算较短区间的结果，然后逐渐将区间扩展到$0 \to n-1$。

我们的解决方案如下：

首先，遍历区间长度。

接下来，将区间$i \to j$分为两个子区间：$i \to k-1$和$k+1 \to j$。

然后，将子区间$i \to k-1$和$k+1 \to j$的得分相加，再加上弹出第k个气球的得分，得到区间$i \to j$的总分。

由于子区间$i \to k-1$和$k+1 \to j$中的气球已经被弹出，所以弹出第k个气球的得分增加量为$nums[k] \times nums[i-1] \times nums[j+1]$。

最后，需要注意子区间的边界处理：因为当前要弹出的气球可能位于区间$i \to j$的边界上，所以我们需要遍历闭区间$[i, j]$内的k值。

最终答案就是$dp[0][n-1]$。

复杂度#

  • 时间复杂度：$O(N^3)$，其中N是列表的长度。
  • 空间复杂度：$O(N^2)$，其中N是列表的长度。

代码#

class Solution:
    def maxCoins(self, nums: List[int]) -> int:
        n = len(nums)
        nums = nums + [1]
        dp = [[0 for _ in range(n+1)] for _ in range(n+1)]

        for i in range(n):
            dp[i][i] = nums[i] * nums[i-1] * nums[i+1]

        for i in range(1, n):
            for l in range(n):
                if i + l == n:
                    break
                r = i + l
                for k in range(l, r+1):
                    dp[l][r] = max(dp[l][r], dp[l][k-1] + dp[k+1][r] + nums[k] * nums[l-1] * nums[r + 1])

        # print(dp)

        return dp[0][n-1]
