Problems & Solutions#

Critical Issues#

1. Vision embeddings extracted from transposed image locations due to X/Y coordinate swap in tissue_positions_list.csv parsing#

Solution: Renamed columns to pxl_row/pxl_col (semantic names), swapped coordinate order to (X, Y) = (pxl_col, pxl_row), deleted vision cache, re-ran experiments

Key Insight: Spatial data requires explicit Y-as-row, X-as-column semantics; 10x scale factor validation (pixel coords × scalefactor = microns) catches coordinate system errors early

2. Pi0_libero checkpoint achieves 0% success on PickPlace despite 96-100% on Coffee/Stack#

Solution: Identified root cause: Pi0_libero is LIBERO-domain finetuned, lacks PickPlace-specific training data (only 10 source demos vs 500 for other tasks); proposed MimicGen→LeRobot→LoRA pipeline

Key Insight: VLA cross-task generalization fails when task domains differ (LIBERO vs PickPlace object/scene layout); finetuning requires 500+ demos expandable via MimicGen from 10 source demos

General Issues#

3. No way to visually inspect VLA policy behavior on PickPlace task to diagnose 0% success rate#

Solution: Built visualize_policy_rollout.py with EGL offscreen rendering, frame-by-frame capture, HUD overlay (policy/step/reward/success), and MP4 export per rollout

Key Insight: Video rollouts reveal behavioral patterns (e.g., random flailing vs directed failures) that success rate alone cannot distinguish

4. Scattered checkpoint paths across multiple HDD_POOL directories making config management fragile#

Solution: Created centralized checkpoints/ directory with task-organized symlinks (bc_rnn/pick_place, vla/pi0_libero, robomimic/lift) and updated all configs to use relative paths

Key Insight: Symlink-based checkpoint organization enables atomic path updates and clear separation between original storage and project references

Learnings#

Key Learnings#

  • Spatial coordinate systems in medical imaging require explicit semantic naming (pxl_row=Y, pxl_col=X) and validation via physical units (pixels × scalefactor = microns); generic pixel_x/pixel_y naming is error-prone
  • Vision foundation model integration in multimodal learning critically depends on spatial alignment; incorrect patch extraction (from transposed locations) can halve performance even with correct architecture
  • VLA models trained on one task domain (LIBERO) show minimal cross-task generalization to different manipulation setups (PickPlace) despite similar action spaces; domain-specific finetuning is essential
  • MimicGen can expand 10 human demonstrations to 500 training examples via automatic variation generation, matching the data scale used in successful VLA finetuning (Coffee: 96%, Stack: 100%)

Practical Learnings#

  • Symlink-based checkpoint management enables version control of path references while preserving original data locations; critical for shared filesystem collaboration
