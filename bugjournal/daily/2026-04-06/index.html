Problems and Solutions#

Key Issues#

1. In HD immune analysis, T cell markers are very sparse (1278/17502 points, average 0.014), while TLS markers are even rarer (242 points). The argument that “multimodal can better capture immune infiltration” does not hold.#

Solution: Changed the narrative in R6 P3 from “multimodal recognition of immune infiltration boundaries” to “balance between clustering and retrieval”. Clustering results showed STAIG had a concentration of 96% in the epithelial domain, while PCA gene expression showed T cell enrichment was 3.8 times that of STAIG. Attention fusion performed more evenly in the boundary region (60/40 epithelial/conductive tissue). Key Discovery: The best strategies for clustering (spatial graph contrast learning) and retrieval (attention fusion) are different. This is a more significant discovery than expected.

2. benchmark_rm_ideal.py writes results to the same summary.csv file by default, causing one experiment’s results to overwrite another.#

Solution: Copied backup files immediately after each experiment was completed (e.g., summary_151673_151674.csv). Used heredoc to manually reconstruct experiments 151673↔151669 because they were overwritten by results from 151674. Key Discovery: Batch experiments need to have a directory for the output in the script, or the results should be aggregated after the pipeline is complete.

3. The conda activate command produces no output in the background, and the output files of all three experiment processes are always empty.#

Solution: Use instead of . This ensures that correct background outputs are written. Key Discovery: The Claude code’s background bash command environment does not inherit the shell’s activation state. Using conda run can ensure environmental isolation.

4. The same experiment was run twice through two different processes (conda activate version + conda run version). Running 6 benchmark tests simultaneously wasted resources.#

Solution: Used ps aux to find the PID of the conda run process and terminated the duplicate 3 processes. Retained the original conda activate version. Key Discovery: When background processes have empty output, it is necessary to confirm whether the process is actually running (using ps aux), rather than directly resubmitting the task.

5. In HD Niche query visualization, query points (red stars) are difficult to visualize and cannot be used for paper presentation.#

Solution: Modified the script: increased the star size from markersize=20 to 26, added a white background marker (size=32), marked all 4 panels, and added arrows and text labels for the first panel. Key Discovery: In paper illustrations, query points require multiple visual cues (color, size, outer ring markers, labels). A single red star is completely unrecognizable against a heatmap background.

Takeaway of Today#

Key Takeaways#

  • Discovered the difference between optimal clustering and retrieval strategies: STAIG’s spatial graph contrast learning is suitable for global domain clustering (average ARI is 0.546), but its strong spatial homogenization leads to excessive focus on the same domain at boundaries (96% epithelial tissue). Attention fusion (60/40 distribution) performs more evenly in boundary microenvironments (60/40 epithelial/conductive tissue).
  • Results from three new RM-IDEAL experiments show that Spearman ρ between donor slices is as high as 0.86, while it reaches 100% between tags @50. However, there is a negative correlation between layers of different donors (ρ=-0.73), indicating systematic differences in the third layer morphology of different donors.
  • In HD colon cancer data, T cell markers (CD3D/CD3E) were expressed at an extremely low rate of 7.3% on 8μm slices. PCA gene expression detected T cell enrichment better than visual and gene fusion methods (2.2 times that of STAIG), indicating that T cell signals are mainly driven by gene expression rather than tissue morphology.
  • Introduction strategy: Directly presenting key numerical results (ARI from 0.26 to 0.55) is more persuasive than general background settings. Using the corresponding result section titles for each paragraph can improve the consistency of paper structure.
