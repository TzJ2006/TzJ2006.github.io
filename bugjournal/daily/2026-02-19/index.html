Key Issues#

1. Regarding PFM embedding techniques for unsupervised clustering methods in ST domain ARIs, there is a lack of direct literature; obtaining accurate data for imaging methods that use only images is very difficult, and no multimodal experiments using only images have ever been conducted.#

Solution: The only clear report on PFM using only image data was found in the SpConTDS paper. The results of stMMC ablation were cited (ARI improvement rate was 4.6%), to quantify the contribution of images, and HEST-1k was used to provide relative rankings for PFM. MILWRM was incorrectly classified as a method using only images, but this was corrected after full-text verification (“gene expression-based”).

Key Insight: There is almost no literature on configurations using only images in the ST domain. This scarcity represents both limitations and opportunities for MIHD to develop baselines using only images. Accurately classifying based on summaries carries systematic risks.

2. STEGO’s training loss was always NaN, and when dividing a dense similarity matrix of 3639×3639 by temperature 0.07, the float32 logsumexp calculation caused overflow issues.#

Solution: A numerically stable implementation was adopted: the sim matrix was truncated to be between (-10, 10), manual logsumexp operations were replaced with F.log_softmax, and L2 normalization was used to standardize input features first.

Key Insight: Large batch data (n_spots approximately 3600) and low temperature values (0.07) may lead to numerical overflow in dense comparison losses. A stable numerical implementation should be chosen during development.

3. The transport/placement stage in 454 scenarios is completely blank. Current demonstration-based generation only triggers the detector at the pre-emptive/reach/upgrade stage.#

Solution: A two-pronged approach was adopted: (1) lowering the detector’s minimum step threshold; (2) using BC-RNN to capture natural errors in later stages—VLA had a success rate of only 0%, so it could not enter the transport/placement stage.

Key Insight: Since the current VLA strategy has a success rate of only 0%, BC-RNN is needed to capture natural errors in later stages. Lowering thresholds and using BC-RNN is the fastest way to fill the gap without waiting for VLA fine-tuning.

4. The second-phase core experiment requires restoring G4 training data, but the success rate of BC-RNN on error scenarios may be close to 0%, so it cannot restore trajectories through automated evaluation.#

Solution: Manual remote operations were used to collect and restore trajectories (at least 200 targets). First, hierarchical processing was performed by error type to independently test representative scenarios, then the entire process was completed.

Key Insight: Although Oracle’s recovery process is automated, its quality is questionable. Remote operation data can provide higher quality and capture natural errors, thus meeting the requirements of the paper.

This was because the basic model was trained with cancer tissue data, and there were subtle morphological differences between brain cortex layers, while fragment-level independent encoding lacked spatial context. There is almost no literature on PFM embedding → unsupervised KMeans → domain ARIs in ST. MIHD is filling this practical literature gap.
  • SCAN self-supervised clustering performed best in DLPFC and multimodal transfer: Feature k-NN mining + clustering consistency loss achieved the best results in image-only clustering (ARI was 0.303, a 20.6% improvement over the baseline of 0.251), better than STEGO and BYOL. SCAN embedding + llava_mlp multimodal fusion further increased ARI to 0.409. Self-supervised refined visual features had transfer advantages in downstream fusion. The average fusion effect using SCAN was the best (+6.5%), while adaln_attention performance declined after using SCAN.
  • BYOL’s unique advantage in H&E domain adaptation: No negative samples required (to avoid ignoring similar layers), insensitive to batch size, STAIG verification showed that BYOL pre-training could improve feature quality and enhance multimodal clustering ARIs—it is the most practical image-only enhancement option for MIHD.
  • A four-step progressive framework for verifying the effectiveness of robot benchmark tests: (1) dataset size and coverage; (2) meaningful classification systems (theoretical basis + empirical consistency + actual discrimination ability); (3) establishing baseline performance; (4) data that can enhance basic models—missing any step will result in failure to convince reviewers.
  • Dependency and performance patterns of PFM tasks: spEMO confirmed that different tasks require different PFMs (spatial recognition最适合使用GPFM/CHIEF, scene retrieval最适合使用UNI2); HEST-1k revealed a logarithmic relationship between PFM performance and model size (Pearson R=0.81); pre-training for pathology was about 8% better in gene expression prediction than ImageNet pre-training.
  • Priority of unsupervised domain adaptation: Across FGVC/Medicine/Radiology/Microscopy domains; within the same domain, unsupervised pre-training is better than direct transfer from general training; feature refinement layers (STEGO/SCAN) are better than direct K-means clustering; spatial graph structure modeling is a key supplement for fine-grained domain recognition.
  • Solutions to the problem of empty transport/placement stages: Lowering the detector’s minimum step threshold (instability reduced from 100 to 50, etc.) and using BC-RNN rolling algorithms (can achieve 60-70% SR in later stages) are the fastest ways to fill the gap. VLA has a success rate of only 0%, meaning that capturing natural errors in later stages cannot rely on oracle recovery. FailSafe and VLA-SCT are plug-ins that do not require training and can be used immediately for M15 comparison experiments after M14 baseline is completed.

Clamp(-10,10) + log_softmax stability treatment is required, and input features must first be normalized using L2 normalization.

  • 验证机器人基准测试有效性的四步渐进框架：（1）数据集规模和覆盖范围；（2）有意义的分类系统（理论基础+经验一致性+实际区分能力）；（3）建立基线性能；（4）能够提升基础模型的数据——缺少任何一步都会导致无法说服审稿人。

  • PFM任务的依赖性和性能模式：spEMO证实不同任务需要不同的PFM（空间域识别最适合使用GPFM/CHIEF，场景检索最适合使用UNI2）；HEST-1k揭示了PFM性能和模型大小之间的对数关系（Pearson R=0.81）；针对病理的预训练在基因表达预测方面比ImageNet预训练好约8%。

  • 无监督领域适应的优先级：跨FGVC/医学/放射学/显微镜领域的应用；在同一领域内，无监督预训练优于从通用训练中直接转移；特征细化层（如STEGO/SCAN）优于直接的K-means聚类；空间图结构建模是精细领域识别的关键补充。

  • 解决运输/放置阶段空白场景的方案：降低检测器的最小步长阈值（不稳定性从100降低到50等），并使用BC-RNN滚动算法（在后期阶段可实现60-70%的检索率）。VLA的成功率为0%，这意味着后期阶段捕捉自然错误不能依赖预测结果。FailSafe和VLA-SCT是无需训练的插件，可以在M14基线完成后立即用于M15比较实验。

  • 密集对比损失（InfoNCE）在处理大批量数据（n-spots约3600）和低温度（0.07）时容易出现数值溢出问题。需要采用clamp(-10,10) + log_softmax稳定化处理，并且输入特征必须先进行L2归一化。

  • 今日要点#

核心要点- The performance limits and literature gaps in ARIs in the pure visual domain: Only ARIs for DLPFC images achieved values of 0.11-0.16 (in contrast, multimodal ARIs ranged from 0.45-0.64). This is because the underlying model was trained with cancer tissue data, resulting in subtle morphological differences between layers of the cerebral cortex. Fragment-level independent encoding lacks spatial context. There is almost no literature on PFM embedding → unsupervised KMeans → ARIs in the pure visual domain. MIHD is filling this real literature gap.#

  • SCAN self-supervised clustering shows excellent performance in DLPFC and multimodal transfer: Feature k-NN mining + clustering consistency loss achieved the best results in image-only clustering (ARIs were 0.303, which was 20.6% higher than the baseline level of 0.251), outperforming STEGO and BYOL. SCAN embedding + llava_mlp multimodal fusion further improved ARIs to 0.409. Self-supervised refined visual features have a transfer advantage in downstream fusion. Using SCAN for average fusion yielded the greatest gain (+6.5%), while adaln_attention showed a decline in performance after using SCAN.
  • BYOL’s unique advantages in adapting to H&E domain: No need for negative samples (to avoid pushing similar layers too far), insensitive to batch size. STAIG has demonstrated that BYOL pre-training can improve feature quality and enhance multimodal clustering ARIs – this is the most practical image-only enhancement option in MIHD.
