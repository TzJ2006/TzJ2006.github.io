Learnings#

Architecture (architecture)#

  • Pre-normalized high-dimensional embeddings lose discriminative power in dimension-agnostic fusion: Teddy 1024d with 460x less per-dim variance than PCA 50d becomes invisible after pipeline L2-normalization. Solution: variance-weighted fusion or PCA-reduce before fusion.
  • ccplan workflow prevents ‘implementing the wrong thing’ through multi-stage pipeline (intent calibration → hypothesis validation → adversarial audit → review gate). Adversarial audit by independent Opus subagent caught 9 issues including fatal concurrency bug before implementation.
  • Custom Hugo theme overrides must be audited against upstream changes — CSS class renames in themes silently break custom templates with no build errors, only runtime layout failures.
  • MimicGen requires patched robosuite with custom APIs (get_bounding_box_half_size, total_size, z_rotation) not in official releases. Patched dependencies must be tracked via unified diff patches to minimize repo size while preserving reproducibility.
  • Region-based UI architecture for concurrent output: allocate independent regions (cursor, buffer, draw logic) per async data source rather than shared state + locks. Applied in MeetingHelper dual audio transcription.
  • Audio pipeline denoising must occur post-capture, pre-ASR: NoiseSuppressor after data.flatten() but before stream.accept_waveform() maximizes effectiveness. Noise fingerprint calibration using initial 0.5s enables stationary noise reduction without pre-trained models.

Debugging (debugging)#

  • Exception handling with bare ’except: continue’ or ’except Exception: pass’ makes debugging extremely difficult. Always capture actual exception, log it, and include in re-raised errors for proper diagnosis.
  • Editable pip installs on NTFS cross-filesystem mounts can leave residual site-packages directories that shadow editable path mappings. Always verify import resolution matches pip show location by checking path and file attributes.
  • Cross-platform line ending differences (CRLF vs LF) create false positives in file comparison. Always use –strip-trailing-cr or -w flags when comparing across Windows/Linux platforms to reveal actual changes.
  • Browser-based visual debugging (DevTools inspect → check applied styles → trace selector mismatch) is more effective for CSS layout issues than server-side file verification.

Tools (tools)#

  • Apple Silicon ASR optimization: CTranslate2 (faster-whisper) requires CUDA; use mlx-based alternatives for Metal GPU acceleration on macOS. Mixed-language ASR requires backend-specific strategies: Whisper needs initial_prompt, Qwen3-ASR auto-detects with language=None.
  • LeRobot 0.4.0 dtype semantics: ‘image’=bytes inline in parquet, ‘video’=MP4 in videos/ directory. For large datasets, always use dtype:‘video’ for efficient MP4 encoding.
  • Hungarian matching is key for converting clustering results to per-class metrics, but masked by permutation-invariant metrics like ARI/NMI. Always align labels before generating confusion matrices or per-layer accuracy.
  • Incremental data pipelines must avoid O(n²) operations like rebuilding entire datasets after each element. Always profile long-running pipelines for hidden O(n²) patterns.

Domain Knowledge (domain)#

  • NeurIPS Datasets & Benchmarks track structure: 9-page main body with Methods in appendix, ML-oriented language. Strong ‘difficulty proof’ essential: Pi0.5 FT 3.7% RSR vs 60-70% normal SR catastrophic gap demonstrates benchmark challenge.
  • Label mixing analysis revealed spatial omics patterns: middle cortical layers (Layer_4 19.3%) have blurred boundaries, outer layers (Layer_1, WM 83-100%) have clear anatomical features. Reflects biological reality of neuron type gradients.
  • Local translation infrastructure: brotli (not brotlicffi) for model downloads, vLLM (includes accelerate) for GPU inference on Linux, transformers fallback on Windows. YAML frontmatter must escape apostrophes in single-quoted strings.
