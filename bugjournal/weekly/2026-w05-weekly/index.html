Learnings#

Debugging (debugging)#

  • Environment configuration (conda environment, clustering libraries) can dominate code-level fixes in ML regression debugging; always verify execution environment matches historical success. Git history comparison with last known good version is often faster than forward-fixing current code; version archaeology should be early in debugging workflow.

Architecture (architecture)#

  • Input dimensionality mismatches (50D PCA vs 3000D raw features) cause catastrophic performance degradation in graph neural networks; always validate feature dimensions. Seemingly minor architectural differences (last layer activation, edge weight direction) accumulate to cause significant divergence when replicating research implementations; exhaustive parameter alignment is critical.

Domain Knowledge (domain)#

  • Clustering method choice (mclust vs kmeans) has first-order impact on spatial transcriptomics evaluation metrics; R integration via rpy2 is often necessary for reproducing bioinformatics paper results.

Tools (tools)#

  • Parallel agent deployment for cross-codebase comparison is highly effective for systematic architectural analysis; deploying agents to explore STAIG and MIHD simultaneously provided comprehensive comparison in one round.
